# 利用上の注意（Disclaimer）

TechBrew は個人向けの RSS ダイジェストツールです。利用者は自己責任でご使用ください。

## RSS 取得について

- 各サイトが公開している **RSS/Atom フィード** のみを利用します
- 取得頻度・件数は `config/sites.yaml` および環境変数で制限してください
- 各サイトの利用規約・robots.txt を尊重し、**個人の情報収集** の範囲で利用してください
- HTML スクレイピングは行いません

## AI 要約（`--ai`）について

- **デフォルトでは無効** です。明示的に `--ai` を指定した場合のみ OpenAI API を呼び出します
- RSS から得たテキストが **OpenAI に送信** されます
- API キーの管理・従量課金・漏洩防止は **利用者の責任** です
- AI 要約は参考情報であり、正確性は保証されません。**必ず元記事を確認** してください
- 記事作者の権利を尊重し、要約のみで済ませず元サイトを訪問することを推奨します

- フォーク・改変されたバージョンの動作についても、各運用者の責任となります

## 推奨される使い方

```bash
go run ./cmd/curator --config config/sites.yaml
```

`--ai` は付けず、RSS の概要とリンクから元記事を読む使い方を推奨します。
